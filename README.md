# 🎭 Random Joke API

A simple Flask API for learning container orchestration with Docker and Kubernetes.

## 📋 Features

- 🃏 Random joke generator endpoint
- ❤️ Health check endpoint for Kubernetes probes
- 🐳 Dockerized for easy deployment
